# Web Demos

Demo website generator for tradie and accountant prospects.

## Tradie Demo
- Generator: `/tradie/generator.html`
- Preview: `/tradie/preview.html`

## Accountant Demo
- Generator: `/accountant/generator.html`
- Preview: `/accountant/preview.html`

## Deploy to Vercel
1. Go to vercel.com → New Project → Import from GitHub
2. Select `Quake95/web-demos`
3. Deploy (no build config needed — static HTML)

Live URLs will be:
- `https://web-demos.vercel.app/tradie/generator.html`
- `https://web-demos.vercel.app/accountant/generator.html`
